# Codex — платформа конспектов

Книжная среда для написания, чтения и распространения учебных конспектов
с интерактивными визуализациями по математике и физике.

---

## Возможности

* Иерархия **книга → глава → конспект** с CRUD на всех уровнях,
  переименованием, перестановкой, дублированием.
* Два рабочих режима: **Конструктор** (редактирование) и **Reader**
  (просмотр с перелистыванием страниц, оглавлением, дроп-капами).
* **20 встроенных анимаций** с настраиваемыми параметрами:
  * Математика: производная, линейные преобразования, ряды Фурье,
    градиентное поле, степень комплексного числа, фазовый портрет,
    метод Ньютона, Монте-Карло, ротор векторного поля, МНК.
  * Физика: баллистика, орбиты Кеплера, интерференция, электрическое
    поле, RLC-цепь, распределение Максвелла, дифракция, маятник,
    лоренцевы факторы, квантовая яма.
* **Экспорт на любом уровне** в форматы:
  * `.codex.json` — для импорта обратно
  * `.html` — самодостаточный одностраничный файл с анимациями
  * `.md` — Markdown
  * `.codex-vite.zip` — отдельный Vite-проект с `run.sh` / `run.bat`
* **Импорт `.codex.json`** с регенерацией ID.
* Локальное хранение в `localStorage` — без бекенда.
* Дизайн в стилистике старой книги: пергамент, серифные шрифты,
  орнаменты, дроп-капы.

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Структура исходников

```
src/
  App.tsx                — переключатель страниц (Library/Constructor/Reader)
  main.tsx               — точка входа
  index.css              — палитра «пергамент + чернила»
  types/                 — модель данных и реестр анимаций
  store/library.ts       — store на useSyncExternalStore + localStorage
  anim/
    render.ts            — все 20 анимаций (canvas 2D)
    AnimationCanvas.tsx  — React-обёртка с DPR/ResizeObserver/RAF
  pages/
    Library.tsx          — список книг + создание/импорт
    Constructor.tsx      — редактор с боковой панелью VS-Code-like
    Reader.tsx           — режим чтения с перелистыванием
  utils/
    exporters.ts         — JSON / MD / HTML / Vite-zip экспорт
```
